# Algorithms & Data Structures
A collection of academic projects focused on classic algorithmic problems and the implementation of data structures.

## Projects included:
- **[Backtracking N-Queens](n-queens-backtracking/):** Implementation of the N-Queens problem using backtracking.
- **[Knapsack](knapsack/):** Classic solution using dynamic programming.
- **[Hanoi Recursive](hanoi-recursive/):** Graphical implementation of the Tower of Hanoi.
- **[N-Queens Graphic](n-queens-graphic/):** Visual approach to the N-Queens problem.
